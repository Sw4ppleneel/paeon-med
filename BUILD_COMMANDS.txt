# BUILD COMMANDS, INSTALL COMMANDS & OUTPUT DIRECTORIES

## FRONTEND (React + Vite)

INSTALL COMMAND:
  npm install

BUILD COMMAND:
  npm run build

OUTPUT DIRECTORY:
  FE/build/

CONFIGURATION:
  - vercel.json: "outputDirectory": "FE/build"
  - vite.config.ts: outDir: 'build'
  - package.json: "build": "vite build"

ENVIRONMENT VARIABLE (Vercel Dashboard):
  VITE_API_BASE_URL=https://your-backend-domain.vercel.app

---

## BACKEND (FastAPI + Python)

INSTALL COMMAND:
  pip install -r requirements.txt

BUILD COMMAND:
  pip install -r requirements.txt

OUTPUT DIRECTORY:
  . (current/root directory)

CONFIGURATION:
  - vercel-backend.json: "outputDirectory": "."
  - Python version: 3.13
  - Entry point: api/index.py
  - Framework: FastAPI

ENVIRONMENT VARIABLE (Vercel Dashboard):
  FRONTEND_URL=https://your-frontend-domain.vercel.app

---

## WHAT YOU NEED TO SETUP

### 1. ENVIRONMENT FILES

Create from templates:
  - .env → copy from .env.example
  - FE/.env.local → copy from FE/.env.local.example
  - FE/.env.production → copy from FE/.env.production.example

### 2. INSTALL VERCEL CLI

  npm install -g vercel
  vercel login

### 3. TEST LOCALLY - BACKEND

  cd /Users/swapneelpremchand/Paeon/Paeon-Med
  pip install -r requirements.txt
  uvicorn main:app --reload --port 8000

Test:
  curl http://localhost:8000/api/health

### 4. TEST LOCALLY - FRONTEND

  cd /Users/swapneelpremchand/Paeon/Paeon-Med/FE
  npm install
  VITE_API_BASE_URL=http://localhost:8000 npm run dev

Open:
  http://localhost:5173 or http://localhost:3000

### 5. BUILD LOCALLY - FRONTEND

  cd FE
  npm run build

Check that FE/build/ directory was created

### 6. DEPLOY BACKEND

  cd /Users/swapneelpremchand/Paeon/Paeon-Med
  vercel --config vercel-backend.json --prod

Note the URL: https://paeon-backend-xxx.vercel.app

### 7. DEPLOY FRONTEND

  cd FE
  vercel --prod

Note the URL: https://paeon-frontend-xxx.vercel.app

### 8. SET ENVIRONMENT VARIABLES

Go to https://vercel.com/dashboard

Backend Project:
  Settings → Environment Variables
  Add: FRONTEND_URL = https://paeon-frontend-xxx.vercel.app

Frontend Project:
  Settings → Environment Variables
  Add: VITE_API_BASE_URL = https://paeon-backend-xxx.vercel.app

### 9. REDEPLOY BOTH

  # Backend
  cd /Users/swapneelpremchand/Paeon/Paeon-Med
  vercel --config vercel-backend.json --prod

  # Frontend
  cd FE
  vercel --prod

### 10. VERIFY

Test backend:
  curl https://paeon-backend-xxx.vercel.app/api/health

Visit frontend:
  https://paeon-frontend-xxx.vercel.app

Check browser console for errors
